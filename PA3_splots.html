PA3 — Fashion-MNIST plots and examples

Run fashionmnist.py to generate the plots. Common output filenames are shown below.

FFN: example classifications

`ffn_examples.png` — one correct + one incorrect example, with labels.

CNN: example classifications

`cnn_examples.png` — one correct + one incorrect example, with labels.

FFN training loss

`training_losses_ffn.png` — training loss over epochs for the FFN.

CNN training loss

`training_losses_cnn.png` — training loss over epochs for the CNN.

Kernels (first conv layer)

`kernels.png` — learned kernels from the first convolutional layer (if generated).

Feature maps

`features.png` — feature maps produced by applying kernels to a sample image (if generated).

To create a single PDF for submission, use the commands in the README.
